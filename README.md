Aegisnyx는 다양한 LLM(대규모 언어모델)을 실제 업무에 활용할 수 있도록 지원하는 AI 애플리케이션 플랫폼입니다.

다음과 같은 고급 기능을 제공합니다.

RAG(Retrieval-Augmented Generation)
웹 검색(Web Search)
코드 실행(Code Execution)
파일 생성(File Creation)
Deep Research
AI Agent

또한 기본 제공되는 50개 이상의 커넥터 또는 MCP(Model Context Protocol)를 통해 다양한 시스템과 연동할 수 있습니다.
